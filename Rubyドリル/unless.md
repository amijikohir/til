# unless
- ifとは逆で、条件式がfalseの場合に処理が実行される
  ※unless式にelsifを指定することはできない
- ifを使用した場合
  ```ruby
    if 条件式  
      条件式がtrueの時に実行する処理  
    else  
      条件式がfalseの時に実行する処理  
    end
  ```
- unlessを使用した場合
  ```ruby
    unless 条件式    
      条件式がfalseの時に実行する処理  
    else    
      条件式がtrueの時に実行する処理  
    end
  ```
